# Brand Collaboration Images

Place your brand collaboration images in this directory.

## Required Images

For each collaboration, you'll need:
- Main image: `[slug].jpg` (e.g., `winspace.jpg`, `cerave.jpg`)
- Gallery images (optional): `[slug]-1.jpg`, `[slug]-2.jpg`, `[slug]-3.jpg`, etc.

## Image Specifications

- Format: JPG recommended
- Main images: Aspect ratio 4:3 works best for masonry grid
- Gallery images: Square (1:1) or landscape (4:3) recommended
- Size: Optimize for web (under 500KB per image)
